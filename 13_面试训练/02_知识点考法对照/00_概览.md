# 02_知识点考法对照 - 概览

> 用「出题者思维」反向训练——知道这个知识点为什么常考，考的是深度还是广度。

## 核心思路

面试官考察一个知识点，通常有三种方式：

| 考法 | 问法举例 | 考的是什么 |
|------|---------|-----------|
| **概念型** | 「是什么？」 | 知不知道，有没有了解过 |
| **原理型** | 「底层是怎么实现的？」 | 理解深度，不是背书 |
| **对比型** | 「和 X 有什么区别？」 | 知识的广度，理解不同方案的权衡 |
| **实战型** | 「你在项目里怎么用的？」 | 能不能落地 |
| **追问型** | 「那如果这样呢？」 | 思维深度，临场反应 |

**这道题「属于哪种」比「答案是什么」更重要。**

## 使用方法

1. 从知识库挑一个核心知识点
2. 按这个模板，分析它可能的考法
3. 先不看答案，自己尝试回答每个考法
4. 对比答案，检查自己在哪个层次卡住了
