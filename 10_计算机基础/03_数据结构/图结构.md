# 图结构

> 图是表达实体间多对多关系的数据结构，是社交网络、地图导航、依赖分析等场景的基石。

---

## 一、图的基本概念

### 1.1 定义与术语

```
有向图（Directed Graph）：        无向图（Undirected Graph）：

  A → B                         A — B
  ↑   ↓                         |\ /|
  D ← C                         | C |
                                |/ \|
                                D — E

G = (V, E)
V = 顶点集合（Vertex）
E = 边集合（Edge）

术语：
- 顶点（Vertex/Node）：图中的节点
- 边（Edge）：连接两个顶点的线
- 有向边：<u, v> 表示从 u 指向 v
- 无向边：(u, v) 表示 u 和 v 互相连接
- 权重（Weight）：边上的数值（距离、成本等）
- 入度（In-degree）：指向该顶点的边数
- 出度（Out-degree）：从该顶点出发的边数
- 自环（Self-loop）：指向自身的边
- 平行边（Parallel edge）：两个顶点之间的多条边
```

### 1.2 图的分类

| 维度 | 分类 | 说明 |
|------|------|------|
| 方向性 | 有向图 / 无向图 | 边是否有方向 |
| 权重 | 有权图 / 无权图 | 边是否携带权重 |
| 连通性 | 连通图 / 非连通图 | 任意两点是否有路径 |
| 自环/平行边 | 简单图 / 多重图 | 是否允许自环和平行边 |
| 层次 | 有环图 / 无环图（DAG） | 是否有环 |

## 二、图的存储方式

### 2.1 邻接矩阵（Adjacency Matrix）

```java
// 使用二维数组存储，matrix[i][j] 表示 i→j 的边
// 无权图：0/1 表示无/有边
// 有权图：0/∞ 表示无，其他值表示权重

int[][] matrix = new int[V][V];
matrix[0][1] = 1;  // A → B
matrix[0][2] = 1;  // A → C
matrix[1][3] = 1;  // B → D
```

| 优点 | 缺点 |
|------|------|
| 判断两点是否有边：O(1) | 空间 O(V²)，稀疏图浪费 |
| 适合稠密图 | 遍历邻接点：O(V) |
| 矩阵运算方便 | 不适合存储大量顶点 |

### 2.2 邻接表（Adjacency List）⭐ 最常用

```java
// 使用 List<List<Integer>> 或 Map<Integer, List<Integer>>
// 每个顶点维护一个邻接点列表

List<List<Integer>> adj = new ArrayList<>();
for (int i = 0; i < V; i++) adj.add(new ArrayList<>());
adj.get(0).add(1);  // A → B
adj.get(0).add(2);  // A → C
adj.get(1).add(3);  // B → D
```

| 优点 | 缺点 |
|------|------|
| 空间 O(V + E)，适合稀疏图 | 判断两点是否有边：O(degree) |
| 遍历邻接点：O(degree) | 删除边较慢 |

### 2.3 对比

| 维度 | 邻接矩阵 | 邻接表 |
|------|---------|--------|
| 空间 | O(V²) | O(V + E) |
| 判断是否有边 | O(1) | O(degree) |
| 遍历所有邻接点 | O(V) | O(degree) |
| 添加边 | O(1) | O(1) |
| 删除边 | O(1) | O(degree) |
| 适用场景 | 稠密图、矩阵运算 | 稀疏图（大多数场景） |

## 三、图的遍历

### 3.1 DFS（深度优先搜索）

**策略**：一条路走到底，碰壁回溯（栈 / 递归）

```java
// 递归实现
boolean[] visited = new boolean[V];
void dfs(int node) {
    visited[node] = true;
    // 处理当前节点
    for (int neighbor : adj.get(node)) {
        if (!visited[neighbor]) {
            dfs(neighbor);
        }
    }
}

// 迭代实现（显式栈）
void dfsIterative(int start) {
    boolean[] visited = new boolean[V];
    Deque<Integer> stack = new ArrayDeque<>();
    stack.push(start);
    while (!stack.isEmpty()) {
        int node = stack.pop();
        if (visited[node]) continue;
        visited[node] = true;
        for (int neighbor : adj.get(node)) {
            if (!visited[neighbor]) stack.push(neighbor);
        }
    }
}
```

```
DFS 遍历顺序（从 A 出发）：
    A
   / \
  B   C
 / \   \
D   E   F

递归 DFS：A → B → D → E → C → F
栈 DFS：  A → C → F → B → E → D（栈是后进先出，所以先压入的会后访问）
```

### 3.2 BFS（广度优先搜索）

**策略**：一层一层扩散（队列）

```java
void bfs(int start) {
    boolean[] visited = new boolean[V];
    Queue<Integer> queue = new LinkedList<>();
    queue.offer(start);
    visited[start] = true;

    while (!queue.isEmpty()) {
        int size = queue.size();
        for (int i = 0; i < size; i++) {
            int node = queue.poll();
            // 处理当前节点
            for (int neighbor : adj.get(node)) {
                if (!visited[neighbor]) {
                    visited[neighbor] = true;
                    queue.offer(neighbor);
                }
            }
        }
    }
}
```

### 3.3 DFS vs BFS 对比

| 维度 | DFS | BFS |
|------|-----|-----|
| 数据结构 | 栈（递归调用栈） | 队列 |
| 空间复杂度 | O(h)，h 为深度 | O(w)，w 为最大宽度 |
| 最短路径 | 不保证 | 无权图最短路径 |
| 适用场景 | 连通性、拓扑排序、回溯 | 最短路径、层级遍历 |
| 是否完全 | 不一定 | 完全遍历 |

## 四、最短路径算法

### 4.1 算法选型

| 算法 | 时间复杂度 | 适用场景 |
|------|-----------|---------|
| BFS | O(V + E) | 无权图最短路径 |
| Dijkstra | O((V + E) log V) | 单源最短路径（非负权重） |
| Bellman-Ford | O(V × E) | 单源最短路径（可有负权重） |
| Floyd-Warshall | O(V³) | 所有节点对最短路径 |
| A* | 启发式 | 实际导航地图 |

### 4.2 Dijkstra 算法

```java
// 小顶堆实现
int[] dijkstra(List<List<int[]>> adj, int src, int V) {
    int[] dist = new int[V];
    Arrays.fill(dist, Integer.MAX_VALUE);
    dist[src] = 0;

    // [距离, 节点]
    PriorityQueue<int[]> pq = new PriorityQueue<>((a, b) -> a[0] - b[0]);
    pq.offer(new int[]{0, src});

    boolean[] visited = new boolean[V];

    while (!pq.isEmpty()) {
        int[] curr = pq.poll();
        int d = curr[0], u = curr[1];
        if (visited[u]) continue;
        visited[u] = true;

        for (int[] edge : adj.get(u)) {
            int v = edge[0], w = edge[1];
            if (dist[u] + w < dist[v]) {
                dist[v] = dist[u] + w;
                pq.offer(new int[]{dist[v], v});
            }
        }
    }
    return dist;
}
```

**Dijkstra 不能处理负权重边的原因**：
- 一旦节点被标记为 visited（距离已确定），就不再更新
- 负权重边可能让已确定的最短距离变得不是最短

### 4.3 Bellman-Ford 算法

```java
// 能处理负权重边，还能检测负权环
int[] bellmanFord(int[][] edges, int V, int src) {
    int[] dist = new int[V];
    Arrays.fill(dist, Integer.MAX_VALUE);
    dist[src] = 0;

    // 松弛 V-1 次
    for (int i = 0; i < V - 1; i++) {
        for (int[] edge : edges) {
            int u = edge[0], v = edge[1], w = edge[2];
            if (dist[u] != Integer.MAX_VALUE && dist[u] + w < dist[v]) {
                dist[v] = dist[u] + w;
            }
        }
    }

    // 第 V 次还能松弛 → 存在负权环
    for (int[] edge : edges) {
        int u = edge[0], v = edge[1], w = edge[2];
        if (dist[u] != Integer.MAX_VALUE && dist[u] + w < dist[v]) {
            throw new RuntimeException("存在负权环！");
        }
    }
    return dist;
}
```

### 4.4 Floyd-Warshall 算法

```java
// 求所有节点对之间的最短路径
int[][] floyd(int[][] graph, int V) {
    int[][] dist = new int[V][V];
    for (int i = 0; i < V; i++)
        System.arraycopy(graph[i], 0, dist[i], 0, V);

    // 中间节点 k，起点 i，终点 j
    for (int k = 0; k < V; k++)
        for (int i = 0; i < V; i++)
            for (int j = 0; j < V; j++)
                if (dist[i][k] != Integer.MAX_VALUE
                    && dist[k][j] != Integer.MAX_VALUE)
                    dist[i][j] = Math.min(dist[i][j], dist[i][k] + dist[k][j]);

    return dist;
}
```

## 五、最小生成树（MST）

### 5.1 概念

连通无向图中，包含所有顶点的树形子图，且边权重之和最小。

### 5.2 Kruskal 算法

**策略**：按边权排序，贪心选取不构成环的边（并查集判断）

```java
int kruskal(int[][] edges, int V) {
    // edges: [u, v, w]
    Arrays.sort(edges, (a, b) -> a[2] - b[2]); // 按权重排序

    UnionFind uf = new UnionFind(V);
    int mstWeight = 0, edgeCount = 0;

    for (int[] edge : edges) {
        int u = edge[0], v = edge[1], w = edge[2];
        if (uf.find(u) != uf.find(v)) { // 不构成环
            uf.union(u, v);
            mstWeight += w;
            edgeCount++;
            if (edgeCount == V - 1) break; // MST 有 V-1 条边
        }
    }
    return mstWeight;
}
// 时间: O(E log E)，空间: O(V + E)
```

### 5.3 Prim 算法

**策略**：从一个起点开始，每次选最小的连接边（小顶堆）

```java
int prim(List<List<int[]>> adj, int V) {
    boolean[] inMST = new boolean[V];
    int[] minDist = new int[V];
    Arrays.fill(minDist, Integer.MAX_VALUE);
    minDist[0] = 0;

    PriorityQueue<int[]> pq = new PriorityQueue<>((a, b) -> a[0] - b[0]);
    pq.offer(new int[]{0, 0}); // [距离, 节点]

    int mstWeight = 0;
    while (!pq.isEmpty()) {
        int[] curr = pq.poll();
        int d = curr[0], u = curr[1];
        if (inMST[u]) continue;
        inMST[u] = true;
        mstWeight += d;

        for (int[] edge : adj.get(u)) {
            int v = edge[0], w = edge[1];
            if (!inMST[v] && w < minDist[v]) {
                minDist[v] = w;
                pq.offer(new int[]{w, v});
            }
        }
    }
    return mstWeight;
}
// 时间: O((V + E) log V)
```

### 5.4 Kruskal vs Prim

| 维度 | Kruskal | Prim |
|------|---------|------|
| 策略 | 按边贪心 | 按顶点贪心 |
| 数据结构 | 并查集 | 小顶堆 |
| 时间复杂度 | O(E log E) | O((V + E) log V) |
| 适合 | 稀疏图 | 稠密图 |

## 六、拓扑排序

### 6.1 概念

对有向无环图（DAG）的所有顶点排一个线性序列，使得对于每条边 <u, v>，u 在序列中排在 v 前面。

### 6.2 Kahn 算法（BFS 入度法）

```java
public int[] topologicalSort(int V, List<List<Integer>> adj) {
    int[] inDegree = new int[V];
    for (int u = 0; u < V; u++) {
        for (int v : adj.get(u)) {
            inDegree[v]++;
        }
    }

    Queue<Integer> queue = new LinkedList<>();
    for (int i = 0; i < V; i++) {
        if (inDegree[i] == 0) queue.offer(i);
    }

    int[] result = new int[V];
    int idx = 0;
    while (!queue.isEmpty()) {
        int node = queue.poll();
        result[idx++] = node;
        for (int neighbor : adj.get(node)) {
            if (--inDegree[neighbor] == 0) {
                queue.offer(neighbor);
            }
        }
    }

    if (idx != V) throw new RuntimeException("存在环！");
    return result;
}
// 时间: O(V + E)
```

### 6.3 应用场景

- **课程安排**：先修课程依赖关系
- **编译顺序**：源文件依赖
- **任务调度**：DAG 中的任务执行顺序
- **包管理**：Maven/Gradle 依赖解析

## 七、并查集（Union-Find）

### 7.1 核心操作

```java
class UnionFind {
    private int[] parent;
    private int[] rank;  // 秩（树高近似）

    public UnionFind(int n) {
        parent = new int[n];
        rank = new int[n];
        for (int i = 0; i < n; i++) parent[i] = i;
    }

    // 路径压缩 + 按秩合并 → 近似 O(1)
    public int find(int x) {
        if (parent[x] != x) {
            parent[x] = find(parent[x]); // 路径压缩
        }
        return parent[x];
    }

    public void union(int x, int y) {
        int px = find(x), py = find(y);
        if (px == py) return;
        // 按秩合并：矮树挂到高树上
        if (rank[px] < rank[py]) parent[px] = py;
        else if (rank[px] > rank[py]) parent[py] = px;
        else { parent[py] = px; rank[px]++; }
    }

    public boolean connected(int x, int y) {
        return find(x) == find(y);
    }
}
// 均摊时间复杂度：O(α(n))，α(n) 是反阿克曼函数，实际 < 5
```

### 7.2 应用

- Kruskal 最小生成树（判断边是否成环）
- 动态连通性（判断两个元素是否在同一集合）
- 网络连通分量

## 八、环检测

### 8.1 无向图环检测

```java
// DFS 检测：如果遇到已访问且不是父节点的邻居，则有环
boolean hasCycle(List<List<Integer>> adj, int V) {
    boolean[] visited = new boolean[V];
    for (int i = 0; i < V; i++) {
        if (!visited[i]) {
            if (dfs(adj, i, -1, visited)) return true;
        }
    }
    return false;
}

boolean dfs(List<List<Integer>> adj, int node, int parent, boolean[] visited) {
    visited[node] = true;
    for (int neighbor : adj.get(node)) {
        if (!visited[neighbor]) {
            if (dfs(adj, neighbor, node, visited)) return true;
        } else if (neighbor != parent) {
            return true; // 遇到已访问的非父节点 → 环
        }
    }
    return false;
}
```

### 8.2 有向图环检测

```java
// 使用三色标记法：白色(未访问) / 灰色(正在访问) / 黑色(已完成)
boolean hasCycle(List<List<Integer>> adj, int V) {
    int[] color = new int[V]; // 0=白, 1=灰, 2=黑
    for (int i = 0; i < V; i++) {
        if (color[i] == 0 && dfs(adj, i, color)) return true;
    }
    return false;
}

boolean dfs(List<List<Integer>> adj, int node, int[] color) {
    color[node] = 1; // 标记为灰色（正在访问）
    for (int neighbor : adj.get(node)) {
        if (color[neighbor] == 1) return true; // 遇到灰色节点 → 环
        if (color[neighbor] == 0 && dfs(adj, neighbor, color)) return true;
    }
    color[node] = 2; // 标记为黑色（已完成）
    return false;
}
```
