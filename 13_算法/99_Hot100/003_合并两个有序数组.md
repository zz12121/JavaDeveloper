## 题干

给你两个按 **非递减顺序** 排列的整数数组 `nums1` 和 `nums2`，以及两个整数 `m` 和 `n`，分别表示 `nums1` 和 `nums2` 中的元素数目。

请你将 `nums2` 合并到 `nums1` 中，使合并后的数组同样按 **非递减顺序** 排列。

### 示例

```
输入：nums1 = [1,2,3,0,0,0], m = 3, nums2 = [2,5,6], n = 3
输出：[1,2,2,3,5,6]
```

我的解题答案：
```java
private static int[] getResultNumArray (int [] a, int alength, int[] b, int blength) {  
    int maxLength = alength + blength -1;  
    int al = alength -1;  
    int bl = blength -1;  
  
    while (al >= 0 && bl >= 0) {  
        if (a[al] > b[bl]) {  
            a [maxLength] = a[al];  
            al --;  
        } else {  
            a [maxLength] = b [bl];  
            bl --;  
        }  
        maxLength --;  
    }  
    while(bl >= 0) {  
        a [maxLength] = b [bl];  
        maxLength --;  
        bl --;  
    }  
    return a;  
}
```
这道题，有几个点：
1.审题不仔细，根本就没看懂使用原数组的条件，然后原数组中有为0的值还认为是什么脏数据。
2.从尾部开始放值是不会导致原数组内容被覆盖，因为数组的长度是2个数组的长度之和，那么原数组预留的长度是可以放另一个数组，那么不管那个位置是放的哪个数组的下标值，都不会抢占原数组的位置，原数组的数据，只会保持不动，或者向后移动。
