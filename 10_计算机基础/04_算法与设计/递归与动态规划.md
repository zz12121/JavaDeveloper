# 递归与动态规划

> 递归是算法的思维方式，动态规划是递归的优化手段。理解"重叠子问题"和"最优子结构"是关键。

---

## 一、递归

### 1.1 递归三要素

```
1. 终止条件（Base Case）：什么时候停下来？
2. 递归过程（Recurrence）：把大问题拆成什么样的小问题？
3. 返回值：每一层需要返回什么给上一层？
```

### 1.2 递归 vs 迭代

| 维度 | 递归 | 迭代 |
|------|------|------|
| 代码风格 | 简洁直观 | 冗长但可控 |
| 栈空间 | O(递归深度) | O(1) |
| 栈溢出风险 | 有（深度过大时） | 无 |
| 适用场景 | 树/图遍历、分治 | 线性/循环处理 |
| 性能 | 函数调用开销 | 无额外开销 |

### 1.3 尾递归优化

```java
// 普通递归：阶乘
long factorial(int n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1); // 递归调用后还有乘法操作，不是尾递归
}

// 尾递归：阶乘（递归调用是最后一步）
long factorialTail(int n, long acc) {
    if (n <= 1) return acc;
    return factorialTail(n - 1, n * acc); // 尾递归
}

// Java 不支持尾递归优化（TCO），但可以用循环替代
long factorialLoop(int n) {
    long result = 1;
    for (int i = 2; i <= n; i++) result *= i;
    return result;
}
```

### 1.4 递归经典问题

```java
// 斐波那契（朴素递归 —— 指数时间复杂度）
int fib(int n) {
    if (n <= 1) return n;
    return fib(n - 1) + fib(n - 2);
}
// 时间: O(2ⁿ)，因为存在大量重复计算

// 记忆化递归 —— O(n)
int fibMemo(int n, int[] memo) {
    if (n <= 1) return n;
    if (memo[n] != 0) return memo[n]; // 命中缓存
    memo[n] = fibMemo(n - 1, memo) + fibMemo(n - 2, memo);
    return memo[n];
}

// 递推（迭代 DP）—— O(n)
int fibDP(int n) {
    if (n <= 1) return n;
    int prev2 = 0, prev1 = 1;
    for (int i = 2; i <= n; i++) {
        int curr = prev1 + prev2;
        prev2 = prev1;
        prev1 = curr;
    }
    return prev1;
}
```

**递归展开图（fib(5)）**：
```
                    fib(5)
                   /      \
              fib(4)      fib(3)
             /     \      /     \
         fib(3)  fib(2) fib(2)  fib(1)
         /   \    / \    / \
     fib(2) fib(1) fib(1) fib(0)

fib(3) 被计算了 2 次，fib(2) 被计算了 3 次
→ 重复子问题！这正是动态规划要解决的
```

## 二、动态规划（Dynamic Programming）

### 2.1 核心概念

**动态规划 = 递归 + 记忆化（空间换时间）**

两个核心条件：
1. **重叠子问题（Overlapping Subproblems）**：子问题被重复计算
2. **最优子结构（Optimal Substructure）**：大问题的最优解包含小问题的最优解

### 2.2 解题四步法

```
Step 1: 定义状态（dp 数组的含义）
Step 2: 状态转移方程（dp[i] 与 dp[i-1] 的关系）
Step 3: 初始化（dp[0], dp[1] 的值）
Step 4: 遍历顺序（从左到右？从上到下？）
```

### 2.3 自顶向下 vs 自底向上

| 维度 | 记忆化搜索（自顶向下） | 递推 DP（自底向上） |
|------|---------------------|-------------------|
| 思维方式 | 自然（从大问题出发） | 反直觉（先算小问题） |
| 实现方式 | 递归 + 缓存 | 循环 + 数组 |
| 空间 | O(n) 递归栈 + O(n) 缓存 | O(n) 数组 |
| 优点 | 只计算需要的子问题 | 无递归栈开销 |
| 缺点 | 可能栈溢出 | 可能计算不需要的子问题 |

## 三、经典 DP 问题分类

### 3.1 一维 DP

#### 爬楼梯

```java
// 状态：dp[i] = 到达第 i 阶的方法数
// 转移：dp[i] = dp[i-1] + dp[i-2]
// 初始化：dp[1] = 1, dp[2] = 2

int climbStairs(int n) {
    if (n <= 2) return n;
    int prev2 = 1, prev1 = 2;
    for (int i = 3; i <= n; i++) {
        int curr = prev1 + prev2;
        prev2 = prev1;
        prev1 = curr;
    }
    return prev1;
}
// 时间: O(n), 空间: O(1)（滚动变量优化）
```

#### 打家劫舍

```java
// 状态：dp[i] = 偷到第 i 家时的最大金额
// 转移：dp[i] = max(dp[i-1], dp[i-2] + nums[i])
// 含义：不偷第 i 家（取 dp[i-1]）vs 偷第 i 家（取 dp[i-2] + nums[i]）

int rob(int[] nums) {
    int prev2 = 0, prev1 = 0;
    for (int num : nums) {
        int curr = Math.max(prev1, prev2 + num);
        prev2 = prev1;
        prev1 = curr;
    }
    return prev1;
}
```

### 3.2 二维 DP

#### 最长公共子序列（LCS）

```java
// 状态：dp[i][j] = text1[0..i-1] 和 text2[0..j-1] 的 LCS 长度
// 转移：
//   text1[i-1] == text2[j-1] → dp[i][j] = dp[i-1][j-1] + 1
//   text1[i-1] != text2[j-1] → dp[i][j] = max(dp[i-1][j], dp[i][j-1])

int longestCommonSubsequence(String text1, String text2) {
    int m = text1.length(), n = text2.length();
    int[][] dp = new int[m + 1][n + 1];

    for (int i = 1; i <= m; i++) {
        for (int j = 1; j <= n; j++) {
            if (text1.charAt(i - 1) == text2.charAt(j - 1)) {
                dp[i][j] = dp[i - 1][j - 1] + 1;
            } else {
                dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
            }
        }
    }
    return dp[m][n];
}
// 时间: O(m × n), 空间: O(m × n) → 可优化到 O(min(m,n))
```

#### 编辑距离

```java
// 状态：dp[i][j] = word1[0..i-1] → word2[0..j-1] 的最少操作数
// 转移：
//   word1[i-1] == word2[j-1] → dp[i][j] = dp[i-1][j-1]（不需要操作）
//   否则 → dp[i][j] = 1 + min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1])
//     dp[i-1][j]   + 1  → 删除 word1[i-1]
//     dp[i][j-1]   + 1  → 插入 word2[j-1]
//     dp[i-1][j-1] + 1  → 替换

int minDistance(String word1, String word2) {
    int m = word1.length(), n = word2.length();
    int[][] dp = new int[m + 1][n + 1];

    for (int i = 0; i <= m; i++) dp[i][0] = i; // 删除全部
    for (int j = 0; j <= n; j++) dp[0][j] = j; // 插入全部

    for (int i = 1; i <= m; i++) {
        for (int j = 1; j <= n; j++) {
            if (word1.charAt(i - 1) == word2.charAt(j - 1)) {
                dp[i][j] = dp[i - 1][j - 1];
            } else {
                dp[i][j] = 1 + Math.min(dp[i - 1][j - 1],
                           Math.min(dp[i - 1][j], dp[i][j - 1]));
            }
        }
    }
    return dp[m][n];
}
```

#### 0-1 背包问题

```java
// 状态：dp[i][w] = 前 i 个物品在容量 w 下的最大价值
// 转移：
//   不选第 i 个：dp[i][w] = dp[i-1][w]
//   选第 i 个：  dp[i][w] = dp[i-1][w-weight[i]] + value[i]
//   dp[i][w] = max(不选, 选)

int knapsack(int[] weight, int[] value, int capacity) {
    int n = weight.length;
    int[][] dp = new int[n + 1][capacity + 1];

    for (int i = 1; i <= n; i++) {
        for (int w = capacity; w >= weight[i - 1]; w--) {
            dp[i][w] = Math.max(dp[i - 1][w],
                       dp[i - 1][w - weight[i - 1]] + value[i - 1]);
        }
    }
    return dp[n][capacity];
}

// 空间优化为一维数组（倒序遍历）
int knapsack1D(int[] weight, int[] value, int capacity) {
    int n = weight.length;
    int[] dp = new int[capacity + 1];

    for (int i = 0; i < n; i++) {
        for (int w = capacity; w >= weight[i]; w--) { // 倒序！保证每个物品只选一次
            dp[w] = Math.max(dp[w], dp[w - weight[i]] + value[i]);
        }
    }
    return dp[capacity];
}
```

### 3.3 区间 DP

#### 最长回文子串

```java
// 状态：dp[i][j] = s[i..j] 是否是回文
// 转移：dp[i][j] = (s[i] == s[j]) && dp[i+1][j-1]
// 边界：dp[i][i] = true（单字符）
//       dp[i][i+1] = (s[i] == s[i+1])

public String longestPalindrome(String s) {
    int n = s.length();
    boolean[][] dp = new boolean[n][n];
    int start = 0, maxLen = 1;

    for (int i = n - 1; i >= 0; i--) {
        dp[i][i] = true;
        for (int j = i + 1; j < n; j++) {
            dp[i][j] = (s.charAt(i) == s.charAt(j)) &&
                        (j - i < 2 || dp[i + 1][j - 1]);
            if (dp[i][j] && j - i + 1 > maxLen) {
                start = i;
                maxLen = j - i + 1;
            }
        }
    }
    return s.substring(start, start + maxLen);
}
```

### 3.4 状态压缩 DP

#### 状态压缩背包

```java
// 用 bitset 表示可达状态
// bitset[j] = true 表示容量 j 可达

int countSubsets(int[] nums, int target) {
    int[] dp = new int[target + 1];
    dp[0] = 1; // 空集
    for (int num : nums) {
        for (int j = target; j >= num; j--) {
            dp[j] += dp[j - num];
        }
    }
    return dp[target];
}
```

## 四、DP 空间优化套路

### 4.1 滚动变量（一维 DP → O(1)）

```java
// 原始：dp[i] 只依赖 dp[i-1] 和 dp[i-2]
int prev2 = dp[0], prev1 = dp[1];
for (int i = 2; i <= n; i++) {
    int curr = prev1 + prev2;
    prev2 = prev1;
    prev1 = curr;
}
```

### 4.2 二维 → 一维

```java
// 原始：dp[i][j] 依赖 dp[i-1][j] 和 dp[i-1][j-w]
// 优化：倒序遍历 j（保证 dp[j-w] 还是"上一行"的值）
for (int i = 0; i < n; i++)
    for (int j = capacity; j >= weight[i]; j--)  // 倒序！
        dp[j] = Math.max(dp[j], dp[j - weight[i]] + value[i]);
```
