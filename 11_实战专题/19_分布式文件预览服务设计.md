# 分布式文件预览服务设计

> 企业内部 OA、合同管理、文档协作等场景，用户需要在线预览各种文件。PDF/Word/Excel/PPT/图片/视频/压缩包，每种格式的预览方式不同，如何设计一个统一的文件预览服务？

---

## 题目一：分布式文件预览服务

### 业务背景

```
企业文件预览场景：

1. 合同管理：上传 Word/PDF 合同 → 在线预览 → 电子签章
2. 财务报销：上传发票图片/报销单 PDF → 在线查看
3. 设计评审：上传 PSD/AI/Sketch → 转成图片预览
4. 代码审查：在浏览器中预览代码文件（语法高亮）
5. 视频会议：上传录屏 MP4 → 在线回放
6. 压缩包查看：上传 ZIP/RAR → 在线查看内部文件列表

支持格式：
  文档：PDF、Word(.doc/.docx)、Excel(.xls/.xlsx)、PPT(.ppt/.pptx)、TXT、CSV
  图片：JPG、PNG、GIF、WebP、SVG、TIFF、PSD
  视频：MP4、WebM、MOV
  音频：MP3、WAV
  其他：ZIP、RAR、代码文件(.java/.py/.js 等)

核心约束：
  - 预览响应时间 < 3 秒（首次打开）
  - 不允许用户直接下载原始文件（安全要求）
  - 支持大文件（500MB 的 PDF 也能预览）
  - 高并发：100+ 人同时预览同一文件
  - 防止越权访问：用户只能预览有权限的文件
  - 水印保护：预览页面叠加用户水印，防止截图泄露
  - 文件转换结果缓存（同一文件不重复转换）
```

---

### 方案对比

| 方案 | 特点 | 支持格式 | 性能 | 推荐场景 |
|------|------|---------|------|---------|
| Office Online Server / LibreOffice | 微软官方 / 开源 | Office 全系列 | 中 | 企业内网、需完美还原排版 |
| kkFileView（开源） | Java + LibreOffice + PDF.js | 文档/图片/视频/压缩包 | 中 | 中小型企业、快速部署 |
| PDF.js（前端） | 纯前端 PDF 渲染 | PDF | 高 | 只需要 PDF 预览 |
| OnlyOffice | Docker 部署的文档编辑器 | Office 全系列 | 中 | 需要在线编辑 + 预览 |
| 自研转图片方案 | 后端转 PDF → 转图片 | 任意格式（取决于转换器） | 高（CDN 加速） | 大型平台、需要深度定制 |
| 阿里云/腾讯云文档预览 | 云服务 API | 全格式 | 高 | 不想自建，愿付费 |

**生产推荐**：中小团队用 **kkFileView**（开箱即用）；大型平台 **自研转图片方案**（可控性强、可集成水印/权限/CDN）；只需 PDF 预览用 **PDF.js**。

#### 文件转换服务

```java
/**
 * 文件转换服务 —— 核心能力
 * 统一入口，根据文件类型分发到不同的转换器
 */
@Service
@Slf4j
public class FileConvertService {

    @Autowired
    private List<FileConverter> converters;
    @Autowired
    private PreviewFileMapper fileMapper;
    @Autowired
    private PreviewConvertTaskMapper taskMapper;
    @Autowired
    private OssService ossService;
    @Autowired
    private StringRedisTemplate redisTemplate;

    /**
     * 转换文件 —— 异步执行
     * 1. 生成转换任务
     * 2. 发送到消息队列
     * 3. Worker 消费消息执行转换
     */
    public Long submitConvertTask(Long fileId, String targetFormat) {
        PreviewFile file = fileMapper.selectById(fileId);

        // 检查是否已有转换结果（相同hash的文件直接复用）
        String cacheKey = "preview:cache:" + file.getFileHash() + ":" + targetFormat;
        String cachedUrl = redisTemplate.opsForValue().get(cacheKey);
        if (cachedUrl != null) {
            // 已有转换结果，直接更新
            file.setPreviewUrl(cachedUrl);
            file.setStatus(2);
            fileMapper.updateById(file);
            log.info("命中转换缓存: fileId={}, hash={}", fileId, file.getFileHash());
            return fileId;
        }

        // 创建转换任务
        PreviewConvertTask task = new PreviewConvertTask();
        task.setFileId(fileId);
        task.setTargetFormat(targetFormat);
        task.setSourcePath(file.getStoragePath());
        task.setStatus(0);
        taskMapper.insert(task);

        // 发送转换消息
        mqTemplate.send("preview:convert",
                new ConvertMsg(task.getId(), fileId, targetFormat));

        return task.getId();
    }

    /**
     * 执行转换（Worker 消费消息后调用）
     */
    @Transactional
    public void executeConvert(Long taskId) {
        PreviewConvertTask task = taskMapper.selectById(taskId);
        if (task.getStatus() != 0) return; // 避免重复消费

        task.setStatus(1);
        task.setWorkerIp(getLocalIp());
        taskMapper.updateById(task);

        PreviewFile file = fileMapper.selectById(task.getFileId());
        long startTime = System.currentTimeMillis();

        try {
            // 选择合适的转换器
            FileConverter converter = selectConverter(file.getFileType(), task.getTargetFormat());

            // 下载源文件
            File sourceFile = ossService.downloadToTemp(file.getStoragePath());

            // 执行转换
            ConvertResult result = converter.convert(sourceFile, task.getTargetFormat());

            // 上传转换结果到 OSS
            String targetPath = "preview/" + file.getFileHash()
                    + "/" + task.getTargetFormat() + "/" + System.currentTimeMillis();
            ossService.upload(new File(result.getOutputPath()), targetPath);

            // 生成预签名 URL
            String previewUrl = ossService.generatePresignedUrl(targetPath, 24 * 3600);

            // 更新文件状态
            file.setStatus(2);
            file.setPreviewUrl(previewUrl);
            file.setPreviewType(task.getTargetFormat());
            file.setConvertTime((int)(System.currentTimeMillis() - startTime));
            fileMapper.updateById(file);

            // 更新任务状态
            task.setStatus(2);
            task.setTargetPath(targetPath);
            task.setFinishTime(LocalDateTime.now());
            taskMapper.updateById(task);

            // 缓存转换结果
            String cacheKey = "preview:cache:" + file.getFileHash() + ":" + task.getTargetFormat();
            redisTemplate.opsForValue().set(cacheKey, previewUrl, 7, TimeUnit.DAYS);

            // 清理临时文件
            sourceFile.delete();
            new File(result.getOutputPath()).delete();

        } catch (Exception e) {
            log.error("文件转换失败: taskId={}, fileId={}", taskId, task.getFileId(), e);
            task.setStatus(3);
            task.setRetryCount(task.getRetryCount() + 1);
            taskMapper.updateById(task);

            // 重试逻辑：最多重试 3 次，指数退避
            if (task.getRetryCount() < 3) {
                mqTemplate.sendDelay("preview:convert:retry",
                        new ConvertMsg(taskId, task.getFileId(), task.getTargetFormat()),
                        (long) Math.pow(2, task.getRetryCount()) * 60 * 1000);
            }
        }
    }

    /**
     * 选择转换器
     */
    private FileConverter selectConverter(String sourceType, String targetFormat) {
        return converters.stream()
                .filter(c -> c.supports(sourceType, targetFormat))
                .findFirst()
                .orElseThrow(() -> new PreviewException(
                        "不支持的转换: " + sourceType + " → " + targetFormat));
    }
}
```

#### 各格式转换器实现

```java
/**
 * 转换器接口
 */
public interface FileConverter {
    boolean supports(String sourceType, String targetFormat);
    ConvertResult convert(File source, String targetFormat) throws Exception;
}

/**
 * Word/Excel/PPT → PDF 转换器
 * 使用 LibreOffice 命令行（生产环境常用）
 */
@Component
public class OfficeToPdfConverter implements FileConverter {

    @Override
    public boolean supports(String sourceType, String targetFormat) {
        Set<String> officeTypes = Set.of("doc", "docx", "xls", "xlsx", "ppt", "pptx");
        return officeTypes.contains(sourceType.toLowerCase()) && "pdf".equals(targetFormat);
    }

    @Override
    public ConvertResult convert(File source, String targetFormat) throws Exception {
        String outputPath = source.getParent() + "/" +
                source.getName().replaceAll("\\.[^.]+$", "") + ".pdf";

        // LibreOffice 命令行转换
        ProcessBuilder pb = new ProcessBuilder(
                "libreoffice", "--headless", "--convert-to", "pdf",
                "--outdir", source.getParent(),
                source.getAbsolutePath()
        );
        pb.redirectErrorStream(true);
        Process process = pb.start();

        // 等待转换完成，最长 120 秒
        boolean finished = process.waitFor(120, TimeUnit.SECONDS);
        if (!finished) {
            process.destroyForcibly();
            throw new PreviewException("Office转PDF超时");
        }

        if (process.exitValue() != 0) {
            throw new PreviewException("Office转PDF失败, exitCode=" + process.exitValue());
        }

        return new ConvertResult(outputPath, "pdf");
    }
}

/**
 * PDF → 图片转换器（用于前端图片查看器渲染）
 * 使用 Apache PDFBox
 */
@Component
public class PdfToImageConverter implements FileConverter {

    @Override
    public boolean supports(String sourceType, String targetFormat) {
        return "pdf".equalsIgnoreCase(sourceType)
                && Set.of("png", "jpg", "webp").contains(targetFormat);
    }

    @Override
    public ConvertResult convert(File source, String targetFormat) throws Exception {
        String outputDir = source.getParent() + "/pages/";
        new File(outputDir).mkdirs();

        PDDocument document = PDDocument.load(source);
        PDFRenderer renderer = new PDFRenderer(document);

        List<String> pagePaths = new ArrayList<>();
        // DPI 控制：150 适合预览（清晰度和文件大小平衡）
        float dpi = 150;

        for (int i = 0; i < document.getNumberOfPages(); i++) {
            String pagePath = outputDir + "page_" + (i + 1) + "." + targetFormat;
            BufferedImage image = renderer.renderImageWithDPI(i, dpi);
            ImageIO.write(image, targetFormat, new File(pagePath));
            pagePaths.add(pagePath);
        }
        document.close();

        return new ConvertResult(outputDir, targetFormat, pagePaths);
    }
}

/**
 * 视频转码器
 * 使用 FFmpeg
 */
@Component
public class VideoTranscodeConverter implements FileConverter {

    @Override
    public boolean supports(String sourceType, String targetFormat) {
        return Set.of("mp4", "mov", "avi", "mkv").contains(sourceType.toLowerCase())
                && "mp4".equals(targetFormat);
    }

    @Override
    public ConvertResult convert(File source, String targetFormat) throws Exception {
        String outputPath = source.getParent() + "/" +
                source.getName().replaceAll("\\.[^.]+$", "_preview.mp4");

        // 转码：降低分辨率 + 码率，适合在线预览
        ProcessBuilder pb = new ProcessBuilder(
                "ffmpeg", "-i", source.getAbsolutePath(),
                "-vf", "scale=1280:-2",           // 宽度 1280，高度自适应
                "-c:v", "libx264",                // H.264 编码
                "-preset", "fast",
                "-crf", "28",                     // 质量参数（越大文件越小）
                "-c:a", "aac",
                "-b:a", "128k",
                "-movflags", "+faststart",        // moov atom 前置（支持边下边播）
                "-y",
                outputPath
        );
        pb.redirectErrorStream(true);
        Process process = pb.start();

        boolean finished = process.waitFor(300, TimeUnit.SECONDS);
        if (!finished) {
            process.destroyForcibly();
            throw new PreviewException("视频转码超时");
        }

        return new ConvertResult(outputPath, "mp4");
    }
}

/**
 * 图片格式转换 + 压缩
 */
@Component
public class ImageConverter implements FileConverter {

    @Override
    public boolean supports(String sourceType, String targetFormat) {
        Set<String> imageTypes = Set.of("jpg", "jpeg", "png", "gif", "bmp", "tiff", "webp");
        return imageTypes.contains(sourceType.toLowerCase());
    }

    @Override
    public ConvertResult convert(File source, String targetFormat) throws Exception {
        String outputPath = source.getParent() + "/" +
                source.getName().replaceAll("\\.[^.]+$", "." + targetFormat);

        // 大图压缩：超过 4096px 等比缩放
        BufferedImage image = ImageIO.read(source);
        if (image.getWidth() > 4096 || image.getHeight() > 4096) {
            Thumbnails.of(source)
                    .size(4096, 4096)
                    .outputFormat(targetFormat)
                    .outputQuality(0.85)
                    .toFile(outputPath);
        } else {
            // 小图直接转格式
            Thumbnails.of(source)
                    .outputFormat(targetFormat)
                    .outputQuality(0.9)
                    .toFile(outputPath);
        }

        return new ConvertResult(outputPath, targetFormat);
    }
}
```

#### 预览访问控制

```java
/**
 * 预览网关 —— 统一处理权限校验、水印、访问控制
 */
@RestController
@RequestMapping("/api/preview")
@Slf4j
public class PreviewController {

    @Autowired
    private PreviewFileMapper fileMapper;
    @Autowired
    private PermissionService permissionService;
    @Autowired
    private OssService ossService;
    @Autowired
    private WatermarkService watermarkService;
    @Autowired
    private PreviewAccessLogMapper logMapper;
    @Autowired
    private StringRedisTemplate redisTemplate;

    /**
     * 获取预览地址
     * 1. 校验用户权限
     * 2. 生成带时效的预览 URL
     * 3. 记录访问日志
     */
    @GetMapping("/url")
    public Result<PreviewUrlVO> getPreviewUrl(@RequestParam Long fileId,
                                               HttpServletRequest request) {
        Long userId = SecurityContext.getCurrentUserId();

        // 1. 校验文件是否存在
        PreviewFile file = fileMapper.selectById(fileId);
        if (file == null) {
            return Result.error("文件不存在");
        }

        // 2. 校验权限
        if (!permissionService.hasPreviewPermission(userId, fileId)) {
            log.warn("无预览权限: userId={}, fileId={}", userId, fileId);
            return Result.error("无预览权限");
        }

        // 3. 检查文件是否已转换完成
        if (file.getStatus() != 2) {
            return Result.error("文件正在转换中，请稍后刷新");
        }

        // 4. 生成带时效的预览签名 URL（10 分钟有效）
        String previewUrl = ossService.generatePresignedUrl(
                file.getPreviewUrl(), 10 * 60);

        // 5. 生成水印参数（用户名 + 时间戳，前端叠加）
        String watermarkText = watermarkService.generateText(userId);

        // 6. 记录访问日志
        PreviewAccessLog accessLog = new PreviewAccessLog();
        accessLog.setFileId(fileId);
        accessLog.setUserId(userId);
        accessLog.setClientIp(getClientIp(request));
        accessLog.setUserAgent(request.getHeader("User-Agent"));
        logMapper.insert(accessLog);

        return Result.success(new PreviewUrlVO(previewUrl, watermarkText, file.getPreviewType()));
    }

    /**
     * 流式预览（大文件分片返回）
     * 用于视频流、大 PDF 的分片加载
     */
    @GetMapping("/stream/{fileId}")
    public void streamPreview(@PathVariable Long fileId,
                               @RequestHeader(value = "Range", required = false) String range,
                               HttpServletResponse response) throws IOException {
        Long userId = SecurityContext.getCurrentUserId();

        // 权限校验
        if (!permissionService.hasPreviewPermission(userId, fileId)) {
            response.setStatus(403);
            return;
        }

        PreviewFile file = fileMapper.selectById(fileId);

        // 大文件从 OSS 分片下载并流式返回
        if (range != null) {
            // Range 请求（视频拖拽进度条时触发）
            long[] rangeValues = parseRange(range, file.getFileSize());
            ossService.streamRange(response, file.getPreviewUrl(),
                    rangeValues[0], rangeValues[1]);
        } else {
            ossService.streamFull(response, file.getPreviewUrl());
        }
    }

    /**
     * 防止下载控制
     * 通过 OSS Bucket Policy 限制：只允许特定 Referer 或签名 URL 访问
     * 前端禁用右键 + 禁用 Ctrl+S
     * 预览 URL 带时效（10 分钟过期）
     * 图片预览返回 WebP 格式（质量降低，不便于二次使用）
     */
}
```

#### 水印服务

```java
/**
 * 水印服务 —— 防止截图泄露
 *
 * 策略：
 * 1. 前端 CSS 水印（简单但不安全，可被 DevTools 删除）
 * 2. 后端图片水印（安全，但增加处理时间）
 * 3. 前端 + 后端结合：后端生成带水印的预览图，前端额外叠加 CSS 水印
 *
 * 生产推荐：后端图片水印 + 前端 CSS 水印双重保护
 */
@Service
public class WatermarkService {

    /**
     * 生成前端水印文本
     */
    public String generateText(Long userId) {
        User user = userService.getById(userId);
        DateTimeFormatter formatter = DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm");
        String time = LocalDateTime.now().format(formatter);
        return user.getName() + " " + time;
    }

    /**
     * 后端图片水印 —— 在转换后的图片上叠加水印
     */
    public File addWatermark(File sourceImage, String watermarkText) {
        try {
            BufferedImage image = ImageIO.read(sourceImage);
            Graphics2D g = image.createGraphics();

            // 设置水印样式
            g.setFont(new Font("SimSun", Font.PLAIN, 28));
            g.setColor(new Color(200, 200, 200, 80)); // 半透明灰色

            // 平铺水印
            int textWidth = g.getFontMetrics().stringWidth(watermarkText);
            for (int y = 100; y < image.getHeight(); y += 200) {
                for (int x = 100; x < image.getWidth() + textWidth; x += textWidth + 200) {
                    // 旋转 30 度
                    g.rotate(Math.toRadians(-30), x, y);
                    g.drawString(watermarkText, x, y);
                    g.rotate(Math.toRadians(30), x, y); // 恢复
                }
            }
            g.dispose();

            File output = new File(sourceImage.getParent(), "wm_" + sourceImage.getName());
            ImageIO.write(image, "png", output);
            return output;
        } catch (IOException e) {
            log.error("添加水印失败", e);
            return sourceImage; // 降级返回原图
        }
    }
}
```

#### Excel 在线预览的特殊处理

```
Excel 预览痛点：
  - 一个 Excel 有 50 个 Sheet，不可能全部转图片
  - 数据量大（10万行），渲染到浏览器会卡死
  - 单元格合并、公式、条件格式难以还原

解决方案：
  1. 小文件（< 5MB，< 1万行）→ 转成 HTML 表格直接渲染
  2. 大文件 → 只转前 100 行预览 + 提示"完整数据请下载查看"
  3. 多 Sheet → 前端 Tab 切换，每个 Sheet 单独请求
  4. 使用 SheetJS (xlsx) 前端库直接解析 Excel → 灵活但格式还原度一般

  生产推荐：小文件用 SheetJS 前端解析（体验好），大文件后端转图片
```

### 高频追问

#### Q1：大文件（500MB PDF）预览如何优化？

```
方案：
1. 分页加载：PDF 转图片后只加载当前页 + 前后各 2 页（虚拟滚动）
2. 懒加载：用户滚动到哪页就加载哪页
3. 预加载：用户看第 1 页时，后台预加载第 2~3 页
4. CDN 加速：转换后的图片通过 CDN 分发
5. WebP 格式：比 PNG 小 30%~50%，减少传输量
6. 分辨率分级：
   - 快速预览：72 DPI（缩略图级别）
   - 高清预览：150 DPI（看清文字）
   - 打印级别：300 DPI（按需加载）
```

#### Q2：如何防止用户通过预览页面下载原始文件？

```
多层防护：
1. 预签名 URL 时效控制（10 分钟过期）
2. OSS Bucket Policy 禁止公开读（所有访问必须带签名）
3. 前端禁用下载：禁用右键菜单、Ctrl+S、F12（治标不治本，但提高门槛）
4. 预览使用低质量版本（图片降低分辨率、视频降低码率）
5. 后端校验 Referer（防止其他网站直接引用预览 URL）
6. 访问频率限制（同一用户 1 分钟内只能获取 N 个预览 URL）
7. 水印追溯（即使被下载/截图，也能追溯来源）

注意：技术上无法 100% 防止下载（截图、录屏都无法阻止）
目标是提高门槛，让普通用户不会随意下载
```

#### Q3：Worker 节点宕机如何处理？

```
问题：文件转换是 CPU 密集型任务，Worker 宕机会导致任务丢失

方案：
1. 转换任务持久化到 DB（已经在做了）
2. 任务状态超时检测：定时任务扫描「处理中超过 10 分钟」的任务
   → 重置为「待处理」→ 重新入队
3. Worker 心跳上报：Worker 每隔 30 秒上报一次
   → 超过 2 分钟没上报 → 标记为下线 → 将其上的任务重新分配
4. 文件去重 + 转换结果缓存：同一 hash 的文件只转换一次
   → Worker 宕机恢复后不会重复转换
5. Worker 水平扩展：通过 MQ 消费者数量自动扩缩
```

### 生产 Checklist

```
□ 文件上传是否有大小限制（如 500MB）
□ 文件格式校验是否检查文件头（魔数）而非仅扩展名
□ 转换任务是否有超时和重试机制
□ 同一 hash 文件是否复用转换结果（避免重复转换）
□ 预览 URL 是否有时效控制（预签名 URL）
□ 是否有权限校验（用户只能预览有权限的文件）
□ 图片预览是否添加水印
□ 大文件是否支持分页/分片加载
□ 转换 Worker 是否有资源限制（CPU/内存/并发数）
□ 是否有转换失败告警（飞书/钉钉通知）
□ OSS 存储是否有生命周期管理（转换临时文件自动清理）
□ 预览访问是否有频率限制（防止接口被刷）
□ 视频预览是否支持 Range 请求（拖拽进度条）
□ Excel 大文件是否做了行数截断（防止前端崩溃）
□ 是否有预览日志审计（谁在什么时候看了什么文件）
```
