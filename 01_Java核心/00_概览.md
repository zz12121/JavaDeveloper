# Java 核心 — 全景地图

> 语言功底是一切的地基。框架会变，Java 核心十年不过时。

---

## 学什么

```
Java 核心
├── 01_基础语法          — 类型、运算、流程控制、字符串、序列化、枚举、final、克隆
├── 02_面向对象          — 封装/继承/多态、抽象类vs接口、内部类、Object类、枚举
├── 03_异常处理          — 异常体系、try-with-resources、异常链
├── 04_泛型              — 类型参数化、通配符、擦除机制、PECS
├── 05_集合框架          — List/Set/Map 各实现原理、HashMap/ConcurrentHashMap 源码
├── 06_IO与NIO           — BIO/NIO/AIO、Channel/Buffer/Selector、零拷贝、序列化
├── 07_注解与反射        — 元注解、APT、反射机制、动态代理、JDK vs CGLIB
├── 08_Lambda与Stream    — invokedynamic、函数式接口、惰性求值、并行流
└── 09_Java新特性        — JDK 8~25 关键演进（Optional/Record/Sealed/虚拟线程/模式匹配）
```

## 怎么学

**顺序**：01 → 02 → 03 → 04 → 05（集合是重点，多花时间）→ 06 → 07（注解+反射，框架的基础）→ 08 → 09

**关键认知**：
- 集合框架是日常开发用得最多的，HashMap/ConcurrentHashMap 必须深入源码
- 泛型擦除是 Java 的设计妥协，理解它才能避开坑
- 注解+反射是所有框架的底层机制（Spring、MyBatis 都依赖它）
- Lambda/Stream 改变了 Java 的编程范式，不是语法糖那么简单

## 和其他域的关系

```
01_Java核心 ←→ 02_并发编程  （集合框架 → 并发容器；多线程 → Thread类）
01_Java核心 ←→ 03_JVM      （类加载 → 反射；字符串 → 常量池）
01_Java核心 ←→ 04_Spring   （注解+反射 → Spring IoC/AOP 的核心机制）
05_集合框架 ←→ 06_中间件    （HashMap → Redis数据结构；List/Set → 消息队列）
```
