## 题干

给定一个未排序的整数数组 `nums`，找出**最长连续序列**的长度。

要求时间复杂度为 O(n)。

### 示例

```
输入：nums = [100,4,200,1,3,2]
输出：4
解释：最长连续序列是 [1,2,3,4]，所以长度为 4。
```

我的解题答案：
```java
private static int getMaxLength(int[] a) {  
    HashSet<Integer> set = new HashSet();  
    int maxLen = 1;  
    for (int i=0; i<a.length; i++) {  
        set.add(a[i]);  
    }  
    for (Integer i : set) {  
        if (!set.contains(i -1)) {  
            //起点  
            int len = 1;  
            int n = i + 1;  
  
            while (set.contains(n)) {  
                len ++;  
                n ++;  
            }  
            maxLen = Math.max(maxLen, len);  
        }  
    }  
    return maxLen;  
}
```
1.在看到题目时，没有任何思路，只是想着暴力破解，那么使用set来处理，判断当前元素是否是一个序列的起点，如果是的话，就从这个元素开始，得到最大的长度，如果不是，就跳过，从序列的起点元素开始。
