# 算法题解题框架

> 刷题不是盲目追求数量，而是掌握一套通用的解题框架。看到题目能快速识别题型，套用模板，这是算法面试的核心能力。

---

## 一、解题方法论

### 1.1 做题流程

```
1. 读题 → 理解输入/输出/约束
2. 举例 → 手动画图，找规律
3. 识别题型 → 套用已知模板
4. 编码 → 先写框架，再填细节
5. 验证 → 用例子走一遍
6. 优化 → 时间/空间能否优化
```

### 1.2 复杂度分析技巧

```
1. 先看嵌套层数：一层循环 O(n)，两层 O(n²)，递归看树高
2. 看数据范围推算法：
   - n ≤ 20          → O(2ⁿ) 回溯/全排列
   - n ≤ 100         → O(n³) Floyd / 三重循环
   - n ≤ 1000        → O(n²) 动态规划
   - n ≤ 10⁵         → O(n log n) 排序 / 二分 / 堆
   - n ≤ 10⁷         → O(n) 线性扫描 / 双指针
   - n ≥ 10⁸         → O(log n) 二分 / 数学公式
3. 递归复杂度 = 节点数 × 每个节点的操作数
```

## 二、核心解题框架

### 2.1 双指针

#### 快慢指针

```java
// 删除有序数组中的重复项
int removeDuplicates(int[] nums) {
    int slow = 0;
    for (int fast = 1; fast < nums.length; fast++) {
        if (nums[fast] != nums[slow]) {
            nums[++slow] = nums[fast];
        }
    }
    return slow + 1;
}

// 移动零
void moveZeroes(int[] nums) {
    int slow = 0;
    for (int fast = 0; fast < nums.length; fast++) {
        if (nums[fast] != 0) {
            int temp = nums[slow]; nums[slow] = nums[fast]; nums[fast] = temp;
            slow++;
        }
    }
}
```

#### 左右指针

```java
// 两数之和（有序数组）
int[] twoSum(int[] nums, int target) {
    int left = 0, right = nums.length - 1;
    while (left < right) {
        int sum = nums[left] + nums[right];
        if (sum == target) return new int[]{left, right};
        else if (sum < target) left++;
        else right--;
    }
    return new int[]{-1, -1};
}

// 反转数组
void reverse(int[] nums, int left, int right) {
    while (left < right) {
        int temp = nums[left]; nums[left++] = nums[right]; nums[right--] = temp;
    }
}
```

#### 滑动窗口

```java
// 无重复字符的最长子串
int lengthOfLongestSubstring(String s) {
    Map<Character, Integer> window = new HashMap<>();
    int left = 0, maxLen = 0;
    for (int right = 0; right < s.length(); right++) {
        char c = s.charAt(right);
        window.put(c, window.getOrDefault(c, 0) + 1);
        // 收缩窗口：直到窗口内没有重复字符
        while (window.get(c) > 1) {
            char leftChar = s.charAt(left);
            window.put(leftChar, window.get(leftChar) - 1);
            left++;
        }
        maxLen = Math.max(maxLen, right - left + 1);
    }
    return maxLen;
}
```

### 2.2 二分搜索

```java
// 模板一：找精确值
int search(int[] nums, int target) {
    int left = 0, right = nums.length - 1;
    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] == target) return mid;
        else if (nums[mid] < target) left = mid + 1;
        else right = mid - 1;
    }
    return -1;
}

// 模板二：找左边界（第一个 ≥ target）
int lowerBound(int[] nums, int target) {
    int left = 0, right = nums.length;
    while (left < right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] < target) left = mid + 1;
        else right = mid;
    }
    return left;
}
```

### 2.3 BFS 模板

```java
// 层序遍历 / 最短路径（无权图）
int bfs(Node start, Node target) {
    Queue<Node> queue = new LinkedList<>();
    Set<Node> visited = new HashSet<>();
    queue.offer(start);
    visited.add(start);
    int steps = 0;

    while (!queue.isEmpty()) {
        int size = queue.size();
        for (int i = 0; i < size; i++) {
            Node curr = queue.poll();
            if (curr.equals(target)) return steps;
            for (Node neighbor : curr.getNeighbors()) {
                if (!visited.contains(neighbor)) {
                    visited.add(neighbor);
                    queue.offer(neighbor);
                }
            }
        }
        steps++;
    }
    return -1;
}
```

### 2.4 回溯模板

```java
// 排列
void permute(int[] nums, List<Integer> path, boolean[] used) {
    if (path.size() == nums.length) { result.add(new ArrayList<>(path)); return; }
    for (int i = 0; i < nums.length; i++) {
        if (used[i]) continue;
        used[i] = true; path.add(nums[i]);
        permute(nums, path, used);
        path.remove(path.size() - 1); used[i] = false;
    }
}

// 组合
void combine(int n, int k, int start, List<Integer> path) {
    if (path.size() == k) { result.add(new ArrayList<>(path)); return; }
    for (int i = start; i <= n; i++) {
        path.add(i);
        combine(n, k, i + 1, path);
        path.remove(path.size() - 1);
    }
}

// 子集
void subsets(int[] nums, int start, List<Integer> path) {
    result.add(new ArrayList<>(path));
    for (int i = start; i < nums.length; i++) {
        path.add(nums[i]);
        subsets(nums, i + 1, path);
        path.remove(path.size() - 1);
    }
}
```

### 2.5 位运算技巧

```java
// 判断奇偶：n & 1
// 交换两数（不用临时变量）：a ^= b; b ^= a; a ^= b;
// 取最低位的 1：n & (-n)
// 去掉最低位的 1：n & (n - 1)
// 判断 2 的幂：n > 0 && (n & (n - 1)) == 0

// 统计 1 的个数（Brian Kernighan 算法）
int countOnes(int n) {
    int count = 0;
    while (n != 0) {
        n &= (n - 1); // 每次去掉最低位的 1
        count++;
    }
    return count;
}
```

## 三、题型识别速查表

| 关键词/特征 | 题型 | 框架 |
|------------|------|------|
| 排好序的数组 | 二分搜索 | left/right 双指针 |
| 找最长/最短/最大/最小子串 | 滑动窗口 | left/right 双指针 |
| 需要原地修改数组 | 快慢指针 | slow/fast |
| 两数/三数之和 | 排序 + 双指针 | left/right |
| 层级遍历 | BFS | 队列 |
| 最短路径（无权） | BFS | 队列 |
| 所有排列/组合/子集 | 回溯 | DFS + 递归 |
| 最长公共子序列 | 二维 DP | dp[i][j] |
| 连续子数组最大和 | 一维 DP | dp[i] / Kadane |
| "Top K" / 最小/最大 | 堆 / 快速选择 | PriorityQueue |
| 需要快速查找/去重 | HashSet / HashMap | — |
| 有依赖关系 | 拓扑排序 | BFS（入度法） |
| 路径/连通性 | 并查集 | Union-Find |
| 按位操作 | 位运算 | & \| ^ ~ << >> |

## 四、刷题建议

### 4.1 按题型刷（推荐顺序）

```
第一轮（基础）：
数组 → 链表 → 栈/队列 → 哈希表 → 二叉树 → 二分搜索

第二轮（进阶）：
滑动窗口 → 双指针 → 回溯 → BFS/DFS → 动态规划

第三轮（专项）：
图 → 堆 → 贪心 → 位运算 → 高级 DP
```
