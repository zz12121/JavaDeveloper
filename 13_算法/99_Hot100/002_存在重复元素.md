## 题干

给定一个整数数组 `nums`，判断是否存在重复元素。

如果存在一个值在数组中出现至少两次，返回 `true`；如果所有元素都不相同，返回 `false`。

### 示例

```
输入：nums = [1,2,3,1]
输出：true

输入：nums = [1,2,3,4]
输出：false
```

#### 我的解题答案：
```java
private static boolean getSameNumResult (int[] a) {  
    if (a == null) {  
        return false;  
    }  
    Set<Integer> numSet = new HashSet<>();  
    for (int i =0; i< a.length; i++) {  
        if (numSet.contains(a[i])) {  
            return true;  
        }  
        numSet.add(a[i]);  
    }  
    return false;  
}
```
### ✅ 解法分析

- **思路**：利用 `HashSet`存储已遍历元素，遇到重复立即返回 `true`，否则遍历完返回 `false`。
- **时间复杂度**：O(n)（HashSet 的 `contains`和 `add`平均为 O(1)）。
- **空间复杂度**：O(n)（最坏情况下无重复，需存所有元素）。
